### How to update code in GitHub
- Add code
```
git add .
```

git add . it's means add all of file change

```
git add [file name]
```
git add [ file name ] add single file

---

- Commit code (comment code)
```
git commit -m "your comment"
```
git commit -m "" it's means comment code

---

- Push code (update code)
```
git push
```
git push it's means update code into GitHub

---
